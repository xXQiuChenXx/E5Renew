# E5Renew Node

Microsoft Graph API tools Build with Typescript
for renew e5 subscription, Office Developer Program.

测试过了，如果用来刷新订阅是没什么问题的。

## Getting Started
1. Clone Repo
   ```
   git clone https://github.com/xXQiuChenXx/E5Renew.git
   cd E5Renew
   ```
2. Run both two commands
   ```
   npm install
   npm start
   ```


Enjoy the code.
